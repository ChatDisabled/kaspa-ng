# kaspa-egui

## Building

To build, please follow Rusty Kaspa [build instructions](https://github.com/kaspanet/rusty-kaspa#getting-started)

## Running Native
```
cargo run
```

## Running Web
```
cargo install trunk
trunk serve
```
Access via [https://localhost:8080](https://localhost:8080)
